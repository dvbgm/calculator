# Калькулятор на C++

Калькулятор на плюсах с веб взаимодействием через библиотеку Crow


## Документация

Чтобы локально развернуть проект нужно:
```
git clone https://github.com/dvbgm/calculator.git
cd calculator
make
```
## Тестирование:
```
cd aleev_calc_fckng_c
make test
```
